About the Project

This project is a Task Manager built with Laravel and Blade, featuring full CRUD functionality to help users create, complete, and manage their tasks effortlessly.

The inspiration behind this project came from my need for an efficient task management tool during my Software Engineering studies at ALX Africa. I wanted to build something practical and useful, not just for myself, but for others who might face similar challenges in task management.

It was created as part of my journey towards obtaining a Software Engineering degree from ALX Africa.
